# DentalCAD 使用手册

## 1. 软件简介

DentalCAD 是基于 Blender 5.0 内核的口腔 CAD/CAM 设计系统，专为口腔种植与修复设计开发。
系统集成了从数据导入到最终加工文件输出的完整数字化工作流，包括：

- 口扫/CBCT 数据导入与对齐
- 单冠 7 步设计流程
- Exocad 格式牙齿库浏览与替换
- 参数化种植基台生成
- 种植手术导板设计与导出
- 全流程工作流面板引导

---

## 2. 系统要求与安装

### 硬件要求

| 项目 | 最低配置 | 推荐配置 |
|------|---------|---------|
| CPU | Intel i5 / AMD Ryzen 5 | Intel i7 / AMD Ryzen 7 |
| 内存 | 8 GB | 16 GB+ |
| 显卡 | OpenGL 4.3 兼容 | NVIDIA RTX 系列 |
| 硬盘 | 2 GB 可用空间 | SSD 10 GB+ |

### 安装步骤

1. 解压 DentalCAD 安装包到任意目录
2. 运行 `dentalcad.exe` 启动程序
3. 首次启动时，在侧边栏 **DentalCAD** 面板设置牙齿库路径

### 牙齿库配置

1. 在侧边栏打开 **牙齿库** 面板
2. 设置 **牙齿库路径** 为 Exocad 格式牙齿库根目录
3. 点击 **刷新** 加载库中的牙型数据
4. 使用下拉框切换不同牙齿库

### 种植体库配置

种植体库位于程序目录下 `Implant_Library` 文件夹。
添加新种植体：编辑 `implant_index.json`，按格式添加条目即可。

## 4. 阶段一：单冠设计

单冠设计采用 7 步工作流，通过侧边栏面板中的 **上一步 / 下一步** 按钮逐步引导。

### 步骤 1：导入预备体 (IMPORT)

**操作：** 点击 `导入预备体` 按钮，选择预备体扫描文件。

- 支持格式：`.stl`、`.obj`、`.ply`
- 导入后自动标记为预备体类型
- 视图自动聚焦到导入模型

> **提示：** 建议使用 STL 格式以确保最佳兼容性。

### 步骤 2：标记边缘线 (MARGIN)

**操作：** 在预备体上标记修复体边缘线。

1. 在预备体表面点击标记关键点（至少 8 个点）
2. 系统自动连接各点生成闭合曲线
3. 边缘线显示为绿色半透明曲线

> **提示：** 点越多，边缘线越精确。推荐 16-24 个点。

### 步骤 3：设定就位道方向 (INSERTION)

**操作：** 设定冠的就位方向。

三种模式可选：
- **Z 轴** - 使用世界坐标 Z 轴方向
- **视角** - 使用当前 3D 视图方向
- **法线** - 使用选中面的平均法线方向

设定后，场景中显示蓝色方向指示线。

### 步骤 4：选择牙型 (TOOTH)

**操作：** 从牙齿库中选择合适的牙型。

1. 确保牙齿库已加载（在牙齿库面板中刷新）
2. 在列表中选择目标牙位
3. 点击 `放置牙冠` 放入场景
4. 牙冠自动定位到边缘线中心并匹配尺寸

**替换功能：** 如需更换牙型，点击 `替换牙冠`（刷新图标），
系统保持位置不变，替换为当前选中的新牙型。

### 步骤 5：适配 (ADAPT)

**操作：** 将牙冠适配到预备体。

- 参数 **水泥间隙**：控制冠与预备体的间距（默认 0.03mm）
- 系统使用 Shrinkwrap 修改器贴合预备体
- 如有边缘线，自动裁剪多余部分

### 步骤 6：邻接调整 (CONTACT)

**操作：** 检测和调整邻牙接触。

- 设置 **接触距离** 阈值（默认 0.05mm）
- 系统显示过紧/过松的接触点
- 手动微调冠的位置以优化咬合

### 步骤 7：导出 (EXPORT)

**操作：** 导出设计好的牙冠。

- 点击 `导出单冠 STL`
- 选择保存路径
- 导出文件可直接用于 CAM 加工

## 5. 阶段二：牙齿库管理

### 库配置

1. 打开侧边栏 **牙齿库** 面板
2. 设置 `牙齿库路径` 为 Exocad 格式库根目录
3. 点击 `刷新` 按钮扫描库目录

### 支持的库格式

- **Exocad 牙齿库标准结构**
  - 每个库一个文件夹
  - 包含 `upperjaw.xml` / `lowerjaw.xml` 元数据
  - 牙型文件为 `.eoff`（二进制 OFF 格式）

### 多库切换

- 使用面板中的下拉菜单选择不同牙齿库
- 切换后自动刷新牙型列表
- 支持动态发现新增的库文件夹

### 牙型浏览

- 列表显示 FDI 牙位号、牙型名称、所属颌弓
- 点击列表项选中后可预览或放置到场景

### 实时替换

在单冠设计的 TOOTH 步骤中：
1. 在牙齿库列表中选择新牙型
2. 点击 `替换牙冠`（刷新图标）
3. 旧冠被移除，新冠导入并保持原始位置
4. 如已标记边缘线，自动重新适配

## 6. 阶段三：种植基台设计

### 参数说明

| 参数 | 说明 | 范围 |
|------|------|------|
| 连接方式 | 基台与种植体的连接类型 | 内六角/外六角/锥形 |
| 平台直径 | 种植体平台连接直径 | 3.0-7.0mm |
| 颈部高度 | 基台颈部光滑段高度 | 0.3-3.0mm |
| 穿龈高度 | 平台到龈缘的过渡高度 | 1.0-8.0mm |
| 穿龈轮廓 | 穿龈区形态 | 直线/凹面/凸面 |
| 顶部直径 | 冠连接端直径 | 2.0-6.0mm |
| 基台角度 | 基台倾斜角度 | 0-25° |

### 生成流程

1. 确保场景中已放置种植体
2. 打开侧边栏 **基台设计** 面板
3. 调整参数至目标值
4. 点击 `生成基台` — 基台自动放置在种植体顶部
5. 修改参数后点击 `更新基台参数` 重新生成
6. 满意后点击 `导出基台 STL`

### 穿龈轮廓类型

- **直线**：从平台直径到顶部直径线性过渡，适用于标准病例
- **凹面**：穿龈区向内收缩（慢-快过渡），适用于软组织较厚区域
- **凸面**：穿龈区向外膨出（快-慢过渡），适用于需要支撑软组织的区域

### 连接方式

- **内六角**：最常用，抗旋转能力强
- **外六角**：传统设计，兼容性好
- **锥形**：莫氏锥度，密封性最佳

## 7. 阶段四：种植手术导板

### 生成导板

1. 确保场景中有颌骨模型（口扫或 CBCT）和种植体
2. 打开侧边栏 **导板设计** 面板
3. 设置导板参数：
   - **导板类型**：牙支持/骨支持/黏膜支持
   - **壳体厚度**：推荐 3.0mm
   - **套筒外径**：匹配所用钻针系统
   - **套筒高度**：推荐 5.0mm
   - **组织面间隙**：推荐 0.1mm
   - **固位钉孔**：建议勾选
4. 点击 `生成手术导板`

### 裁剪导板边界

1. 在 **高级功能** 区域设置 `裁剪余量`（默认 10mm）
2. 点击 `裁剪导板边界`
3. 系统自动计算种植区域并裁剪多余部分

### 套筒碰撞检测

点击 `检测套筒碰撞`，系统检查各套筒间距：
- 绿色提示：间距正常
- 黄色警告：显示具体碰撞对和间距

> **注意：** 需要至少 2 个种植体才能进行碰撞检测。

### 分段导板

对于大范围手术导板，可分段制作：
1. 设置 `分段数`（2-4 段）
2. 点击 `分段导板`
3. 系统沿最长水平轴均匀切割

### 质量检查

点击 `检查导板`，系统检测：
- 网格流形性（非流形边数）
- 最小尺寸（避免过薄区域）
- 面数（建议 < 50 万面）

### 导出

点击 `导出导板 STL`，选择保存路径。
导出文件可用于 3D 打印（SLA/DLP 推荐）。

## 8. 种植规划

### 数据导入

#### 口扫导入
- 菜单：`牙科设计 > 导入口扫数据`
- 支持 STL/OBJ/PLY 格式
- 导入后自动设置半透明显示

#### CBCT 导入
- 菜单：`牙科设计 > 导入 CBCT`
- 支持 STL/OBJ/PLY 骨面模型
- 支持 DICOM 文件夹（需 pydicom + numpy）

#### 模型对齐
- 同时选中两个模型（如口扫+CBCT）
- 执行 `融合对齐`
- 优先使用 ICP 算法；如未安装 Open3D，回退到质心对齐

### 种植体放置

1. 在种植体库面板中选择种植体型号
2. 将 3D 光标移至目标位置
3. 点击 `放置种植体`
4. 种植体自动放置在光标位置

### 轴向调整

- **手动调整**：选中种植体后用旋转工具 (R) 调整
- **设定轴向** (dental.set_implant_axis)：精确输入角度
- **吸附到骨面** (dental.snap_to_surface)：自动贴合最近骨面

### 骨量测量

1. 选中目标位置的对象
2. 执行 `骨量测量`
3. 系统沿 Z 轴上下射线检测骨面
4. 显示可用骨高度（mm）

## 9. 常见问题与故障排除

### Q: 牙齿库刷新后列表为空

**A:** 检查以下几点：
1. 牙齿库路径是否正确（应指向包含子库文件夹的根目录）
2. 子库文件夹中是否包含 `.eoff` 牙型文件
3. 查看控制台是否有 XML 解析错误提示

### Q: 布尔运算失败

**A:** 常见于导板生成和裁剪步骤：
1. 确保输入网格是闭合流形
2. 尝试先对模型执行 `平滑网格` 或 `填充孔洞`
3. 检查控制台输出的具体错误信息

### Q: 导入预备体后视图没有聚焦

**A:** 确保存在 3D 视口窗口。如果从脚本执行，系统会自动查找 3D 视口进行聚焦。

### Q: 基台生成报错 "未找到种植体"

**A:** 请先通过种植规划流程放置种植体，或导入种植体模型文件。

### Q: CBCT 导入报错 "缺少依赖包"

**A:** 需要安装 pydicom 和 numpy：
```
pip install pydicom numpy
```

### Q: STL 导出后文件很大

**A:** 
1. 使用 `平滑网格` 工具降低面数
2. 在导出前检查面数（导板检查功能可显示）
3. 建议导板面数控制在 50 万以内

### Q: 种植体吸附方向异常

**A:** 吸附功能基于骨面法线方向。如果模型法线不正确：
1. 选中骨面模型
2. 进入编辑模式
3. 全选 → `Mesh > Normals > Recalculate Outside`
